A catastrophic Ultimate Skill that unleashes multiple simultaneous Disintegration fields across a wide area. It is an amplified mass-cast version of Disintegration.

## Information

| Type | Ultimate Skill |
|------|----------------|
| Obtain Cost | 2000000 MP |
| Points to Master | 1000 |
| Mode | Hold |

---

## Obtaining

Mega Disintegration is an Ultimate Skill that cannot normally be obtained.

- This skill is not obtainable through Reincarnation
- This skill is not obtainable through Skill Reroll
- It exists as an experimental skill

---

## Usage

### [Active - Hold] Mega Disintegration - Unleashes a mass Disintegration field across a 25-block radius around the user.

- Hold Time: 10 seconds
- Damage: 1500
- On release after full charge, a/multiple Disintegration Entity/s is/are deployed at the target location/s or entity position/s
- If released early, the skill is cancelled and the attack fails

---

## Notes

- This skill is **not intended for standard player use**
- The skill is not obtainable by any means
- It is not fully balanced for normal gameplay
- Designed primarily as a fun to use Ultimate Skill
