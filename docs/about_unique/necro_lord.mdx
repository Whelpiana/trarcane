A forbidden necromantic Unique Skill that grants dominion over death, soul-binding artifacts, and partial transcendence beyond mortality.

## Information

| Type | Unique Skill |
|------|--------------|
| Obtain Cost | 75000 MP |
| Points to Master | 1000 |
| Passive | Toggle |

---

## Obtaining

Necro Lord is a Unique Skill that can be obtained through:

- Reincarnation
- Skill Reroll

---

## Usage

### [Active - Press] Lich Book Creation -Creates a Necromancy tome used for unlocking undead-related abilities.

- Costs: 50000 Magicules
- Grants a Lich Book item to the user
- Using the Lich Book grants the user the following Magics:
  - Create Lesser Undead
  - Create Greater Undead
  - Curse
  - Curse Bind
- Requires possession of Necro Lord skill to use

---

### [Active - Press] Phylactery Creation - Create of a soul-bound resurrection artifact.

- Costs: 500000 Magicules
- Creates a Phylactery bound to the user's UUID
- Plays enchantment table sound on creation
- Displays confirmation message upon success

Phylactery Properties:
- Bound to owner's UUID
- Stores EP
- Cannot be used by other players

---

### Resurrection Effect (Phylactery)
Triggers automatically upon death if conditions are met:

- Requires valid ownership binding
- Requires ≥ 250000 EP durability
- Consumes the Phylactery on activation
- Restores full health
- Removes all status effects
---

### [Active - Press] Wraith Form - Grants the user flight.

---

## Mechanics

- Phylactery  acts as a conditional resurrection
- Resurrection only triggers when soul energy threshold is met
