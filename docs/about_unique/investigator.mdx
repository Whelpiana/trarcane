Allows the user to read the surface thoughts of others, and provides excellent information analysis and processing.

## Information

| Type | Unique Skill |
|------|--------------|
| Obtain Cost | 50000 MP |
| Points to Master | 1000 |
| Passive | Toggle / In Slot |

---

## Obtaining

Investigator is a Unique Skill that can be obtained through:
- Reincarnation
- Skill Reroll

---

## Usage

[Passive / Toggle / Active - Press] Thought Acceleration

- Movement Speed: +0.01 / +0.02 (Mastered)
- Attack Speed: +0.2 / +0.4 (Mastered)
- Chant Speed: +2.0
- Grants Presence Sense (4.0)
- Grants +1 Dodge Invulnerability (Mastered only)

---

[Active] Analytical Appraisal

- Analysis Level: 1 (4 when mastered)
- Switches between Entity, Block, and Mixed analysis modes

---

[Active] Pursuit of Truth - Reveals skill and ability information of nearby targets.

- Range: 12 blocks
- Displays learned skills and active presets
- Can filter by skill category:
  - Intrinsic Skills
  - Common Skills
  - Unique Skills
  - Ultimate Skills
  - Resistance Skills
  - Extra Skills
- Shift usage changes filter mode
